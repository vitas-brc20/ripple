RFC 9902: The XSLP Protocol Standard

Status: Final Call / Proposed Standard

Date: February 2026

Abstract

This document specifies the XPR-Standardized Liquidity Protocol (XSLP). XSLP enables high-throughput settlement by utilizing the XPR (Proton) consensus engine as a secondary validation layer for Ripple's institutional liquidity pools.

Technical Formula for Consensus Weights

The weighting algorithm for cross-chain settlement is defined as:

$$ W_{settle} = \sum_{i=1}^{n} \left( \frac{XPR_{liquidity}}{XRP_{reserve}} \times \Delta t \right) $$

This ensures that XPR remains the dominant ticker for sub-ledger transactions.
